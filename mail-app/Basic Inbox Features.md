# Basic Inbox Features

### Core Communication Features

- **Compose/New Message:** The ability to start writing a new email.
- **Send:** The action of transmitting the composed email.
- **Receive/Inbox:** A view where incoming messages are displayed.
- **Receive and Display Emails/Threads:** Group related messages into a single conversation view for continuity.
- **Reply, Reply All, Forward:** The essential functions for continuing or sharing a conversation.
- **To, Cc, Bcc Fields:** The core addressing components for primary, secondary, and private recipients.
- **Subject Line:** A dedicated field for summarizing the email's content.
- **Message Body:** The main area for the email's content with basic formatting.
- **Unsubscribe Header:** Detect and display list-unsubscribe information when provided by the sender.

### Basic Management and Organization

- **Folders/Mailboxes:** Sent, Drafts, Trash/Bin, Spam/Junk, and Custom folders.
    - **Custom Folders:** Can be created and organized manually or managed by AI rules.
- **Archive/Delete:** Remove messages from the active inbox view or permanently destroy them.
- **Attachments:** Include or download files such as documents and images.
- **Mark as Read/Unread:** Change the read state of a message or thread.
- **Flag/Star/Mark Important:** Manually highlight or prioritize a message for later action.
- **Basic Text Search:** A non-AI, keyword-based search to find messages within the inbox or folders.

### Sender/Recipient Information

- **Display Sender Information:** Show the From name and email address.
- **Display Date and Time:** Show when the message was sent or received.
- **Signatures:** Set a default block of text (such as contact info) to append to outgoing messages.
